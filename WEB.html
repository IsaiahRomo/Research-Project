Software Licensing Project

Brief Description

Software Licensing is a document that provides the use of software to a certain extent while providing guidelines to the user on what they can do. It also allows the user to have the right to own one or more copies of it without violating copyrights. Some software licenses that are free allow to use of the software to no extent other that are paid softwares have restrictions and guidelines that you must follow.

Purpose

Software Licensing allows people to own software that companies make without breaking any copyrights of owning it as long as you follow the guidelines. This allows companies to make money and provide a software that is reliable and still up to date. Otherwise people wouldn’t be able to get certain software without getting copyright.
